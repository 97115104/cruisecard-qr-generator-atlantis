# Atlantis Cruise Card

## Overview
The Atlantis Cruise Card is a web application designed to generate QR codes for cruise cards. It features a user-friendly interface that allows users to input their details and generate a QR code that can be scanned for boarding.

## Features
- **Responsive Design**: The application is optimized for both desktop and mobile devices.
- **Dynamic QR Code Generation**: Utilizes the `qrcodejs` library to generate QR codes based on user input.
- **Customizable Fields**: Users can add custom fields to their cruise cards.
- **Loading Screen**: A visually appealing loading screen with animations while the QR code is being generated.

## Technologies Used
- **HTML**: Structure of the web application.
- **CSS**: Styling and layout, including responsive design features.
- **JavaScript**: Functionality for QR code generation and dynamic content updates.
- **Google Fonts**: Custom fonts for improved typography.

## Usage
1. Open the `index.html` file in a web browser.
2. Fill in the required fields to create your cruise card.
3. Click the "Generate" button to create your QR code.
4. The QR code will be displayed on the screen, ready for scanning.

## Customization
- Modify the CSS variables in the `:root` section to change the color scheme and styles of the application.
- Add or remove fields in the HTML to customize the information displayed on the cruise card.

## Conclusion
The Atlantis Cruise Card application provides a simple yet effective way to generate cruise cards with QR codes, enhancing the boarding experience for users.
